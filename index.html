Why?

We built Senna to be a lightweight plugin that can turn any static site into a fast and responsive app without adding frameworks or rewriting tons of code.

Blazing Fast

Our clean API's can turn your static site into a blazing fast single-page application in just a couple clicks.

Minimal Bandwidth

By cacheing reusable resources like CSS and Script files, Senna significantly lowers your user's bandwidth.

Incredibly Versatile

No need to switch frameworks. Senna is unobtrusive and can be easily added to your existing HTML or JavaScript.

Using Senna

There are many ways to use Senna. You can install the NPM depenency, link the CDN, or even download the full package with example files. Here is one of the easy ways to get started with Senna in a couple clicks.

1. Link

First, add Senna to your document using CDN.

<head>
    <script src="https://cdn.jsdelivr.net/senna.js/2.1.1/senna-min.js">
</head>

2. Add

Then, include the Senna attributes into your body element

<body data-senna data-senna-surface>
    ...	
</body>

3. Run

Now navigating through different pages of your app to see how fast it is!

How does it work?

We built Senna to be a lightweight plugin that can turn any static site into a fast and responsive app without adding frameworks or rewriting tons of code.

1

LOAD

Your app will load all the background resources on the initial page load and caches them for future use.

2

NAVIGATE

A link is clicked and Senna populates the new url even though it is not fully loading a new page.

3

REQUEST

The request is made but Senna utilizes the cached resources before pulling any new elements.

4

PAINT

Quickly, Senna finds what content is new to the requested page and paints only the new content to the screen.
